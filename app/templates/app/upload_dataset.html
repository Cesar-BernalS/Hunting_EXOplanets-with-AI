Instrucciones para el Archivo
Formato Requerido
  • Archivos CSV o Excel (.xlsx, .xls)
  • Máximo 10MB de tamaño
  • Primera fila debe contener los nombres de las columnas
Columnas Esperadas
  • name - Nombre del candidato
  • orbital_period - Período orbital (días)
  • planetary_radius - Radio planetario (R⊕)
  • classification - Clasificación (opcional)
Consejo

Si tu archivo no tiene todas las columnas requeridas, el sistema intentará mapear automáticamente las columnas disponibles. Puedes editar la información después de subir el archivo.
